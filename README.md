# MAX! to MQTT Bridge Add-on Repository

This repository contains the MAX! to MQTT Bridge Home Assistant Add-on.

## Installation

1. Add this repository to your Home Assistant Add-on Store:
   `https://github.com/pwurbs/max2mqtt`
2. Install the **MAX! to MQTT Bridge** add-on.
3. Configure the add-on according to the [Readme](README.md).
4. Read more detailled information about the add-on in the [Documentation](max2mqtt/DOCS.md).
